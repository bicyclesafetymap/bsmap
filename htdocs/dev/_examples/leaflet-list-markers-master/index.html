Leaflet List Markers

A Leaflet Control for listing visible markers/features in a interactive box

Other useful stuff for Web Mapping...

Features

  • List Visible Markers
  • Customize item text by marker attribute
  • Locate marker on item click
  • Customize style icon and arrow for items list
  • List GeoJSON features coming soon
